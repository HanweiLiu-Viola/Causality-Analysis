# Tutorial Notebooks

Run the notebooks in this order for the Section 7 tutorial:

| Order | Notebook | What it does |
|---|---|---|
| 1 | `00_startup.ipynb` | Checks required tools and starts the pinned Docker container. |
| 2 | `01_make_bids_data.ipynb` | Generates five simulated subjects and writes a BIDS-compatible EEG dataset. |
| 3 | `02_run_connectivity_analysis.ipynb` | Runs the random-model effective-connectivity example and saves figures. |
| 4 | `03_run_workflow.ipynb` | Previews the Snakemake workflow and verifies expected outputs. |

The following notebooks are retained as extension or development material and
are not required for the main tutorial:

- `04_connectivity_benchmark.ipynb`
- `05_statistical_tests.ipynb`
